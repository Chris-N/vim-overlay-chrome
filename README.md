# vim-overlay-chrome
vim overlay chrome extension

Quick keyboard navigation for any website. 

## Hotkeys
+ Shift + /: VIM hotkey guide**
+ j: scroll down
+ k: scroll up
+ d: page down
+ u: scroll up
+ g: jump to top
+ Shift + g: jump to end
+ m: set bookmark
+ ': jump to bookmark
+ CMD + /: toggle on/off VIM
